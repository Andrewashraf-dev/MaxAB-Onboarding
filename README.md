# Onboarding System – MaxAB

## Overview

This project is a web-based **Employee Onboarding System** designed to automate and streamline the process of collecting, validating, and managing new employee information before their first working day.

New joiners receive a secure onboarding link where they can submit their personal details and upload required documents. The system stores this data in a backend database and generates structured Excel reports for HR teams.

---

## Tech Stack

* **Frontend:** React
* **Backend:** Java (Spring Boot)
* **File & Excel Processing:** Apache POI
* **Data Storage:** Relational Database
* **Reporting:** Excel (.xlsx)

---

## Features

* Self-service onboarding portal for new employees
* Secure form with required field validation
* Photo and document upload support
* Automatic data persistence in database
* Excel report generation (one sheet per employee)
* Downloadable consolidated reports for HR

---

## System Flow

1. A new employee receives an onboarding link before their first working day.
2. The employee submits personal information and uploads required documents.
3. Data is validated and stored in the backend database.
4. Apache POI generates structured Excel files with one sheet per employee.
5. HR downloads reports for internal processing and integration with other systems.
